# FC SEGMENT AI

Lovable UI company website for Agentic AI products. Built with React + Vite + Tailwind CSS.

## Scripts

- dev: `npm run dev`
- build: `npm run build`
- preview: `npm run preview`

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

Open the URL printed in the terminal.

## Tech
- React + Vite
- Tailwind CSS
- React Router
